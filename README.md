# 💍 Wedding Invitation Website

A beautiful, responsive wedding invitation website built with HTML, CSS, and JavaScript. Hosted on GitHub Pages for free!

## 🌐 Live Website

Visit your wedding invitation at:
**https://ranaasamah.github.io/wedding-invitation/**

## ✨ Features

- 🎨 **Beautiful Design** - Modern, elegant gradient design
- ⏱️ **Countdown Timer** - Live countdown to the wedding day
- 📅 **Event Details** - Date, time, and venue information
- 📍 **Location Map** - QR code and Google Maps integration
- 🎉 **Event Schedule** - Timeline of wedding events
- 🍽️ **Menu Display** - Wedding feast details
- ✍️ **RSVP Form** - Guests can confirm attendance
- 📱 **Fully Responsive** - Works on all devices
- ⚡ **Fast Loading** - Optimized performance

## 📋 How to Customize

### 1. Edit Names and Details
Open `index.html` and replace:
- `GROOM FIRST NAME` and `GROOM SECOND NAME`
- `BRIDE FIRST NAME` and `BRIDE SECOND NAME`
- Wedding date, time, and venue information
